The Project

This project was created by Nathan Pogue (BSc & HBA), David Chen (BMSc), Ahmed Mohamed (BSc), Tara Chen (BMSc), and Ajith Rahugnanam (HBSc) to detect and classify seven different types of skin cancers and lesions. Using the MNIST HAM10000 dataset of 10,015 dermascopic photos, DermaGO utilizes a convolutional neural network to analyze and predict the classification of user-uploaded skin cancer and lesion images.

Using this kernel for reference and guidance, DermaGO's model is able to correctly predict and classify images at an 80.42% test accuracy. Using the Keras InceptionV3 prebuilt architecture paired with different convolution and pooling layers, DermaGO's model was trained on over 8,000 images to make these predictions.

Please see our research paper for more details.
